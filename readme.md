# Thunderbirdで、Content-Language: ja-JP にするダミー辞書
1. jajp.xpiをダウンロードして、Thunderbird アドオンとテーマ → 辞書、を選んで「ファイルからアドオンをインストール」でインストール
2. 設定 → 編集 → スペルチェック → 辞書の言語で、日本語をチェック
3. メッセージ作成で「スペル」ボタンのプルダウンメニューで、日本語が選択されていることを確認。
